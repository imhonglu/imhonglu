# 정범구

## 개발

- 개발자가 신경 쓰지 않아도 되는 강력한 타입 추론을 좋아합니다.
- 바퀴를 항상 다시 만드는 것을 좋아합니다.

## 최근 관심사

- 최근 JSON Schema 구현체를 만들어 보고 있어요.
  - StructuralValidation 항목만 마무리하고 끝낼지 고민입니다.
- JSON Schema 기반으로 모든 구간에서 통용되는 프로젝트를 만들어보고 싶어요.

## 입문 동기

- 군대에서 스마트폰을 들고 PHP 코딩을 접한 이후로 흥미가 생겨서
- 전역 이후 국비지원 과정으로 자바를 배워 개발에 입문하게 됐습니다.
